= Shortcode and Widget Options =

**General**

* Enable Paging? - For `[ldd_deliveries_widget_list]`
	* Disable – Don't display paging
	* Enable – display paging before and after post entries
	* Before – display paging only before post entries
	* After – display paging only after post entries

**Testing**

* Debug Mode - Bypass Ajax controller to handle posts_to_import directly for testing purposes.
* Posts to Import - A CSV list of post ids to import, like "1,2,3".
* Skip Importing Posts - A CSV list of post ids to not import, like "1,2,3".
* Import Limit - Useful for testing import on a limited amount of posts. 0 or blank means unlimited.

**Compatibility & Reset**

* Export Settings – These are your current settings in a serialized format. Copy the contents to make a backup of your settings.
* Import Settings – Paste new serialized settings here to overwrite your current configuration.
* Remove Plugin Data on Deletion? - Delete all Legal Document Deliveries - Core data and options from database on plugin deletion
* Reset to Defaults? – Check this box to reset options to their defaults


== Installation ==

= Requirements =

* PHP 5.3+ [Read notice](https://aihrus.zendesk.com/entries/30678006)

= Install Methods =

* Through WordPress Admin > Plugins > Add New, Search for "Legal Document Deliveries - Core"
	* Find "Legal Document Deliveries - Core by Aihrus"
	* Click "Install Now" of "Legal Document Deliveries - Core by Aihrus"
* Download [`ldd-deliveries.zip`](http://downloads.wordpress.org/plugin/ldd-deliveries.zip) locally
	* Through WordPress Admin > Plugins > Add New
	* Click Upload
	* "Choose File" `ldd-deliveries.zip`
	* Click "Install Now"
* Download and unzip [`ldd-deliveries.zip`](http://downloads.wordpress.org/plugin/ldd-deliveries.zip) locally
	* Using FTP, upload directory `ldd-deliveries` to your website's `/wp-content/plugins/` directory

= Activation Options =

* Activate the "Legal Document Deliveries - Core" plugin after uploading
* Activate the "Legal Document Deliveries - Core" plugin through WordPress Admin > Plugins

= Usage =

1. Edit options through WordPress Admin > Settings > Legal Document Deliveries - Core 
1. Process posts via WordPress Admin > Tools > Legal Document Deliveries - Core

= Upgrading =

* Through WordPress
	* Via WordPress Admin > Dashboard > Updates, click "Check Again"
	* Select plugins for update, click "Update Plugins"
* Using FTP
	* Download and unzip [`ldd-deliveries.zip`](http://downloads.wordpress.org/plugin/ldd-deliveries.zip) locally
	* Upload directory `ldd-deliveries` to your website's `/wp-content/plugins/` directory
	* Be sure to overwrite your existing `ldd-deliveries` folder contents
