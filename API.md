# API Legal Document Deliveries - Core

The [Legal Document Deliveries - Core plugin](http://wordpress.org/plugins/ldd-deliveries/) comes with its own set of actions and filters, as described below.

## Actions

* `ldd_deliveries_update`

	Update helper.
		
* `ldd_deliveries_settings_add_help_tabs`

	Customize settings page help tabs.

* `ldd_deliveries_styles`

	Styles tie-in.

* `ldd_deliveries_scripts`

	Scripts tie-in.

## Filters

* `ldd_deliveries_validate_settings`

	Validate settings helper. [example](https://gist.github.com/michael-cannon/5833768)

* `ldd_deliveries_version`

	Version tracking for settings. [example](https://gist.github.com/michael-cannon/5833774)

* `ldd_deliveries_sections`

	Alter section options. [example](https://gist.github.com/michael-cannon/5833757)

* `ldd_deliveries_settings`

	Alter setting options. [example](https://gist.github.com/michael-cannon/5833757)

* `ldd_deliveries_display_setting`

	Customize display settings content.

* `ldd_deliveries_widget_options`

	Customize widget options.

* `ldd_deliveries_defaults`

	Customize defaults.

* `ldd_deliveries_defaults_single`

	Customize defaults for single instance.
